The fuel parameters allocate vehicle miles traveled for each of the four road vehicle types that RPAT represents – auto, light truck, bus, and heavy truck (truck) – to different fuel types. The remaining proportion of fuel not accounted for in these categories is assumed to be gasoline:

  • PropDiesel: The proportion of the fleet that uses diesel.
  • PropCng: The proportion of the fleet that uses compressed natural gas.
  • GasPropEth: The average ethanol proportion in gasoline sold.
  • DieselPropBio: The average biodiesel proportion in diesel sold.

This file is used in the calculations of fuel consumption. This file can be used to test alternative fuel scenarios by varying the shares of non-gasoline fuels.
