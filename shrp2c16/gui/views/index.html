                              Set up the Base Scenario

                              The base scenario is named Base. It is used to identify the scenario and to create a directory to hold the actual input/output files.
                              Please type a scenario name.
                              Scenario Name



                              Copy from:



                              A model run is based on a number of input files. The Base Scenario represents a "business as usual" future. It is the scenario against which you will compare different policy scenarios to understand their impact.

                              To create the Base Scenario, click on the Create Base Scenario button below and go to the Scenarios tab. In the Scenarios tab, you can edit input files based on the assumptions you want to make or policies you want to test. Once you are done editing the input files, you will have the ability to run the future scenario and look at the results.

                              To create a new scenario, simply go back to this page and press the Create New Scenario button.
